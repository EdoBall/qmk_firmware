# IQS5xx ATI Target と ATI Compensation 調整ガイド

## 1. 基本的な理解

### ATI Targetとは
- **目的**: 全チャンネルが目指すべき基準カウント値
- **範囲**: 通常100-1000（推奨：500-700）
- **影響**: 高い値 = 高感度、低い値 = 低感度

### ATI Compensationとは
- **目的**: 各チャンネルの物理的差異を補正する値
- **範囲**: 0-255
- **理想値**: 100-150（中央付近）
- **自動設定**: ATI TargetとATI Cの組み合わせで自動決定

## 2. 調整の基本原則

### 2.1 適切なATI Compensationの範囲
```
理想的な範囲: 80-180
警告範囲: 50未満 または 200以上
危険範囲: 20未満 または 230以上
```

### 2.2 調整の優先順位
1. **ATI C** の調整（粗調整）
2. **ATI Target** の調整（微調整）
3. **個別チャンネル** の調整（最終調整）

## 3. 段階的調整手順

### ステップ1: 初期状態の確認
```
1. 全チャンネルのATI Compensationを読み取り
2. 平均値、最大値、最小値を確認
3. 極端に異なるチャンネルを特定
4. ノイズレベルを測定
```

### ステップ2: ATI C による粗調整
```
ATI Compensationが全体的に高い場合（180以上）:
→ ATI C を 1-2 増加

ATI Compensationが全体的に低い場合（80未満）:
→ ATI C を 1-2 減少

変更後は必ずATIを再実行して結果を確認
```

### ステップ3: ATI Target による微調整
```
感度が不足している場合:
→ ATI Target を 50-100 増加

感度が過敏すぎる場合:
→ ATI Target を 50-100 減少

目標: タッチ時のデルタ値が 200-800 の範囲
```

### ステップ4: 個別チャンネル調整
```
特定チャンネルのATI Compensationが極端な場合のみ:
→ 個別ATI C調整を使用
→ ±1-3 の範囲で調整
```

## 4. 実践的な調整テクニック

### 4.1 感度テスト方法
```
軽いタッチテスト:
- 4つのチャンネルの境界に指を軽く置く
- 全て同程度の反応があることを確認
- デルタ値: 150-300 が理想

強いタッチテスト:
- 各チャンネルの中心を強く押す
- デルタ値: 500-1000 が理想
- 飽和（カウント値が下がる）がないことを確認
```

### 4.2 ノイズ耐性の確認
```
ノイズフロア測定:
- 無タッチ状態でのカウント値変動を観測
- ±5以下が理想、±10以下が許容範囲
- 近接しきい値はノイズの3倍以上に設定
```

### 4.3 温度変動対応
```
温度テスト:
- 低温時（0-10°C）と高温時（40-50°C）でテスト
- ATI Compensationの変動を確認
- ±20以内の変動が理想
```

## 5. トラブルシューティング

### 5.1 ATI Compensationが極端に高い（200以上）
```
原因と対策:
1. ATI C値が低すぎる → ATI Cを1-2増加
2. ATI Targetが高すぎる → ATI Targetを50-100減少
3. ハードウェア問題 → 配線、GNDプレーン確認
```

### 5.2 ATI Compensationが極端に低い（50未満）
```
原因と対策:
1. ATI C値が高すぎる → ATI Cを1-2減少
2. ATI Targetが低すぎる → ATI Targetを50-100増加
3. センサー容量が大きすぎる → ハードウェア見直し
```

### 5.3 チャンネル間のばらつきが大きい
```
許容範囲: ±30以内
対策:
1. 個別ATI C調整を使用
2. ハードウェアの均一性確認
3. 配線長の統一
4. GNDプレーンの最適化
```

### 5.4 感度が不安定
```
症状と対策:
1. タッチ検出が不安定 → ATI Targetを上げる
2. 誤検出が多い → しきい値を上げる
3. 温度で変動 → ATI Compensationを中央寄りに
```

## 6. 最適化のワークフロー

### 6.1 初期設定
```c
// 推奨初期値
ATI_C = 3;           // 中間値から開始
ATI_Target = 600;    // やや高めから開始
```

### 6.2 反復調整プロセス
```
1. ATI実行
2. ATI Compensation確認
3. 感度テスト
4. 必要に応じてATI C調整
5. ATI実行
6. 感度微調整（ATI Target）
7. 最終テスト
```

### 6.3 検証項目
```
✓ 全チャンネルのATI Compensation: 80-180
✓ チャンネル間ばらつき: ±30以内
✓ 軽いタッチのデルタ値: 150-300
✓ 強いタッチのデルタ値: 500-1000
✓ ノイズレベル: ±5以下
✓ 温度変動: ATI Compensation ±20以内
```

## 7. 高度な調整テクニック

### 7.1 環境別最適化
```
乾燥環境（湿度30%未満）:
- ATI Targetを10-15%上げる
- しきい値を少し下げる

高湿度環境（湿度80%以上）:
- ATI Targetを5-10%下げる
- しきい値を少し上げる
```

### 7.2 材料別調整
```
厚いオーバーレイ（2mm以上）:
- ATI Targetを20-30%上げる
- ATI Cを1-2下げる

薄いオーバーレイ（0.5mm以下）:
- ATI Targetを10-20%下げる
- しきい値を上げる
```

### 7.3 アプリケーション別設定
```
高精度トラッキング用途:
- ATI Compensation: 120-140（狭い範囲）
- 高いATI Target（700-800）

低消費電力用途:
- ATI Compensation: 100-160（広い範囲許容）
- 中程度のATI Target（500-600）
```
